## Form Handling вҖ” PHP Approach

No third-party service. Form submissions handled entirely on Hostinger.

```
User submits form
       вҶ“
ContactForm.astro (POST action="/contact.php")
       вҶ“
contact.php (lives in /public вҖ” served at obviusdigital.ca/contact.php)
       вҶ“
PHP mail() function
       вҶ“
Delivered to @obviusdigital.ca business email
```

`contact.php` validates input server-side, sends the email, and returns a JSON response. The form handles success/error display client-side.

## CSS Architecture

- `global.css` вҖ” Reset, CSS variables, body, typography, utility classes
- `layout.css` вҖ” Header, navigation, footer, page grid
- `components.css` вҖ” All component-level styles per m0t.WEB.1.1 section order

```css
--color-red: #C0392B;
--color-black: #0A0A0A;
--color-white: #FFFFFF;
--color-grey-light: #F5F5F5;
--color-grey-mid: #999999;

```
